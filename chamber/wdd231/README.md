# Ruwa Town Chamber of Commerce Project

## Overview
The Ruwa Town Chamber of Commerce project aims to create a comprehensive online platform for local businesses and services in Ruwa. The website will feature a directory of chamber members, a home landing page, and future pages for joining the chamber and discovering more about its offerings.

## Project Structure
```
wdd231
├── chamber
│   ├── index.html          # Home (landing) page for the Chamber of Commerce
│   ├── directory.html      # Directory of chamber members showcasing local businesses and services
│   ├── join.html           # Future page for users to join the Chamber of Commerce
│   ├── discover.html       # Future page to help users discover more about the chamber
│   ├── images              # Directory for storing images used in the project
│   ├── scripts
│   │   ├── home.js         # JavaScript for home page functionality
│   │   ├── directory.js     # JavaScript for directory page functionality
│   │   └── navigation.js    # JavaScript for navigation functionality
│   └── styles
│       ├── home.css        # CSS styles specific to the home page
│       └── directory.css    # CSS styles specific to the directory page
├── styles
│   ├── normalize.css       # CSS to normalize styles across different browsers
│   ├── small.css           # CSS for smaller screen sizes
│   └── large.css           # CSS for larger screen sizes
└── README.md               # Documentation for the project
```

## Features
- **Home Page (index.html)**: Includes a consistent header, navigation, footer, hero image, call to action, current events section, weather section, and company spotlights.
- **Directory Page (directory.html)**: Showcases local businesses and services.
- **Join Page (join.html)**: Planned for future implementation to allow users to join the chamber.
- **Discover Page (discover.html)**: Planned for future implementation to help users learn more about the chamber and its offerings.
